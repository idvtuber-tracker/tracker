01

Overview

IDVTuber Tracker ("we", "our", "the project") is a non-commercial fan community project that monitors and displays publicly available livestream analytics from Indonesian Virtual YouTuber (VTuber) channels on YouTube. This Privacy Policy explains what data we collect, how we use it, and your rights regarding that data.

We are committed to operating transparently and in full compliance with the YouTube API Services Terms of Service and all applicable developer policies. By using this dashboard, you acknowledge that you have read and understood this Privacy Policy.

✓

IDVTuber Tracker is strictly non-commercial. We collect no personal data, display no advertising, and generate no revenue of any kind.

02

YouTube API Services

IDVTuber Tracker uses the YouTube Data API v3, provided by Google LLC, to collect publicly available livestream analytics data from YouTube channels. By using this dashboard, you are also subject to Google's Privacy Policy and the YouTube Terms of Service.

→

Google's Privacy Policy: http://www.google.com/policies/privacy
YouTube Terms of Service: https://www.youtube.com/t/terms

All data collected via the YouTube API is publicly available information — the same data any person could view by watching a YouTube livestream. No private, authenticated, or restricted YouTube data is accessed at any time.

03

Data We Collect

IDVTuber Tracker collects only the following publicly available data returned by the YouTube Data API v3 during active or upcoming livestreams:

Channel ID
Unique YouTube channel identifier (e.g. UCxxxxxx)
Channel Name
Public display name of the YouTube channel
Video ID
Unique YouTube video/stream identifier
Video Title
Public title of the livestream at time of collection
Concurrent Viewers
Number of viewers watching the stream at time of snapshot
Like Count
Cumulative public like count at time of snapshot
Comment Count
Cumulative public comment count at time of snapshot
Stream Status
"live" or "upcoming" as reported by YouTube API
Stream Start Time
Scheduled and actual stream start timestamps
Collection Timestamp
Date and time when each data snapshot was recorded
✓

IDVTuber Tracker does not collect, access, or store any personally identifiable information about YouTube users, channel subscribers, comment authors, or dashboard visitors. No usernames, profile pictures, email addresses, IP addresses, or any other personal data are collected at any time.

04

How We Use Data

Data collected via the YouTube API is used exclusively for the following purpose:

To generate and display analytics visualizations on this public dashboard, including concurrent viewer trend charts, peak viewer counts, engagement metrics, and historical data tables for each tracked livestream. This data is displayed solely for informational and fan community engagement purposes.

Collected data is stored in a private PostgreSQL database hosted on Supabase and is never used for advertising, user profiling, machine learning training, resale, or any commercial purpose whatsoever.

05

Data Sharing

IDVTuber Tracker does not sell, rent, license, or share any collected data with any third party for any purpose. The only parties with access to the raw database are the project operators.

Aggregated, anonymized analytics (such as viewer trend charts) are displayed publicly on this dashboard. This data contains no personally identifiable information and reflects only publicly available YouTube metrics.

The following third-party services are used to operate this project:

YouTube Data API v3
Data source — Google LLC. Subject to Google Privacy Policy.
Supabase
Database hosting — Supabase Inc. Data stored in private PostgreSQL instance.
GitHub Pages
Dashboard hosting — GitHub Inc. Static files served publicly.
GitHub Actions
Automation — GitHub Inc. Used for running the tracker workflow.
06

Data Retention & Deletion

YouTube API data is retained for a rolling 90-day window. A scheduled database job automatically and permanently deletes all records older than 90 days on a daily basis at 03:00 UTC.

This means no individual stream analytics snapshot is stored for longer than 90 days from its collection date. Data is never archived or transferred to any other storage system beyond our primary database.

✓

To request deletion of any specific data, contact us at the email address in the Contact section below. All verified deletion requests will be fulfilled within 7 business days.

07

Cookies & Tracking Technologies

IDVTuber Tracker does not use cookies, local storage, session storage, tracking pixels, browser fingerprinting, web beacons, or any other technology to collect information from visitors' devices or browsers.

This dashboard is a fully static website served via GitHub Pages. It contains no analytics scripts (such as Google Analytics), no advertisement scripts, no social media tracking widgets, and no third-party JavaScript of any kind beyond the Chart.js visualization library loaded from a public CDN for rendering viewer trend charts.

✓

No information is stored on or retrieved from your device at any point during your visit to this dashboard.

08

Your Rights & Revoking Access

IDVTuber Tracker uses only a server-side API key to access the YouTube Data API. No individual user OAuth authentication or user tokens are used or stored. As such, there is no per-user API access to revoke through this dashboard.

In accordance with YouTube API Services Developer Policies, you may revoke access that any Google-authenticated application has to your YouTube data by visiting the Google Security Settings page:

→

Google Security Settings: https://security.google.com/settings/security/permissions

If you are a content creator whose channel is tracked by this project and wish to request removal from the tracker, please contact us using the details in the Contact section below. We will remove your channel from tracking and permanently delete all associated data within 7 business days.
