01

Overview

IDVTuber Tracker ("we", "our", "the Dashboard") is a non-commercial fan project that monitors publicly available YouTube livestream analytics for Indonesian VTuber organisations. This Privacy Policy explains what data we collect, how it is stored, and how it is deleted.

By using this Dashboard you agree to the practices described in this policy.

02

Data We Collect

We collect only publicly available, non-personalised YouTube livestream statistics retrieved via the YouTube Data API v3. Specifically:

  • Video ID and video title
  • Channel ID and channel name
  • Concurrent viewer count
  • Like count and comment count
  • Stream status (live / upcoming / ended)
  • Scheduled start time and actual start time
  • Timestamp of when the data point was collected
ℹ

All collected data is publicly visible on YouTube. We do not access any private or authenticated user data.

03

Data We Do Not Collect

We do not collect any of the following:

  • Personal information about Dashboard visitors (no name, email, IP address, or cookies)
  • YouTube account information or authenticated user data
  • Private or unlisted video data
  • Comments content or individual commenter identities
  • Financial or subscription data
ℹ

This Dashboard has no user accounts, no login system, and sets no cookies. Visiting the Dashboard is entirely anonymous.

04

Storage & Retention

Collected API data is stored in a PostgreSQL database hosted on Supabase. Data is retained for a rolling 30-day window. A scheduled database job automatically deletes rows older than 30 days on a daily basis.

Retention period: 30 days

Database host: Supabase (PostgreSQL)

Deletion schedule: Daily at 03:00 UTC via pg_cron

The static Dashboard pages (HTML files) are hosted on GitHub Pages. These pages display aggregated statistics only and do not contain any personal data.

05

Deletion

Individual data records older than 30 days are deleted automatically via a pg_cron scheduled job running daily at 03:00 UTC.

Users may also request deletion of any stored data by contacting us at idvtuber.tracker@gmail.com. Upon receiving a verified deletion request, all associated data will be permanently removed within 7 business days.

⚠

Because we collect only public YouTube statistics and no personal data, deletion requests are typically for the removal of a specific channel's data from our database. Please include the channel name or YouTube channel URL in your request.
