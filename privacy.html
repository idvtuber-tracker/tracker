Description of Service

IDVTuber Tracker is a non-commercial, fan community analytics dashboard that collects and displays publicly available livestream metrics from Indonesian Virtual YouTuber (VTuber) channels. The Dashboard shows:

  • Real-time and historical concurrent viewer counts during livestreams
  • Like counts and comment counts collected during active streams
  • Interactive viewer trend charts over time
  • A searchable archive of all tracked streams and their analytics

All data displayed on this Dashboard is publicly available information sourced from the YouTube Data API v3. This project has no affiliation with, endorsement by, or relationship with any talent agency, content creator, or YouTube LLC.
